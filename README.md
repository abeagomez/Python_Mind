# Python_Mind
Simple Python 2.7 Master Mind

Just run it and play in terminal!

You have 9 attempts.

"V" indicates how many numbers are right but in wrong position.

"T" indicates how many numbers are right and in the right position.

Range for numbers: 1-9

Game example:

>>>Guess

>>>__1 4 5 6__

>>>[1, 4, 5, 6]  V: 1   T: 0

>>>Guess

>>>__9 8 6 5__

>>>[9, 8, 6, 5]  V: 0   T: 3

>>>Guess

>>>__9 8 6 1__

>>>[9, 8, 6, 1]  V: 0   T: 3

>>>Guess

>>>__9 8 6 5__

>>>[9, 8, 6, 5]  V: 0   T: 3

>>>Guess

>>>__9 8 6 4__

>>>[9, 8, 6, 4]  V: 0   T: 3

>>>Guess

>>>__9 8 6 1__

>>>[9, 8, 6, 1]  V: 0   T: 3
